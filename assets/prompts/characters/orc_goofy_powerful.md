# Goofy Powerful Orc Reference Prompt

Use this prompt to generate the single approved reference sprite for the base
orc before requesting any sprite sheet.

## Character Identity
The orc is goofy yet powerful: broad shoulders, big hands, small tusks, slightly
confused expression, and a chunky fantasy silhouette. It should look dangerous
enough to be an enemy, but playful enough to fit a bedtime adventure game.

## Style
- Top-down / three-quarter 2D pixel art
- Strong contrast against grass, stone walls, and Theo's blue pyjamas
- Saturated green skin with warm shadow tones
- Chunky readable pixels, no painterly blur
- Transparent background

## Required Output
- Single full-body sprite, facing down/front
- Centered in a 64x64 transparent canvas
- Simple club or rough weapon allowed, but keep the silhouette readable
- No text, no frame border, no background scene

## Negative Prompt
No horror gore, no realistic monster anatomy, no tiny unreadable details, no
photorealism, no extra characters, no background.
