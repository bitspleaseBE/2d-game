# Theo Pyjamas Reference Prompt

Use this prompt to generate the single approved reference sprite for Theo before
requesting any sprite sheet.

## Character Identity
Theo is a small, brave child hero wearing cozy blue pyjamas. He should feel
sleepy but determined: soft rounded face, tousled brown hair, oversized pajama
top and trousers, tiny slippers, and a bedtime-adventure silhouette.

## Style
- Top-down / three-quarter 2D pixel art
- Strong contrast against green grass and dark stone walls
- Chunky readable pixels, clean silhouette, no painterly blur
- Cute bedtime fantasy mood, not modern streetwear
- Transparent background

## Required Output
- Single full-body sprite, facing down/front
- Centered in a 64x64 transparent canvas
- No weapon in this reference
- No text, no frame border, no shadow that extends beyond the canvas

## Negative Prompt
No armor, no jeans, no sneakers, no adult proportions, no realistic rendering,
no soft airbrush, no extra characters, no background scene.
